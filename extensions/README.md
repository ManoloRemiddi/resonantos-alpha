# Extensions

OpenClaw extension packages.

## Active Extensions

| File/Directory | Purpose |
|----------------|---------|
| `r-awareness.js` | Contextual document injection (before_turn/after_turn hooks) |
| `r-memory.js` | Conversation compression hooks |
| `shield-gate/` | Security gate implementation |
| `coherence-gate/` | Coherence management |
| `gateway-lifecycle.js` | Gateway lifecycle management |
| `usage-tracker/` | Usage tracking |

## Configuration

Extension configs are centralized in `config/extensions/`.

## Installation

Extensions are installed to `~/.openclaw/extensions/` by the install script.
