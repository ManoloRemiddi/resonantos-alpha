## Section 4: Recommendations

### Immediate Actions (P0)

1. **Add typosquatting detection**
   ```python
   # Use Levenshtein distance against known domains
   PROTECTED_DOMAINS = ["anthropic.com", "openai.com", "google.com"]
   if levenshtein(domain, protected) <= 2:
       BLOCK("Potential typosquatting")
   ```

2. **Normalize unicode before scanning**
   ```python
   import unicodedata
   text = unicodedata.normalize('NFKC', text)
   text = text.replace('\u200b', '')  # zero-width
   ```

3. **Add recursive base64 decoding**
   ```python
   while is_base64(text):
       text = decode_base64(text)
       scan(text)
   ```

### High Priority (P1)

4. Add patterns for:
   - `os.system()`, `subprocess.`, `eval()`, `exec()`
   - File/env var exfiltration requests
   - Social engineering authority claims
   - Debug mode requests

5. Improve A2A agent card scanning:
   - Scan name field separately with injection patterns
   - Detect delimiter injection (`---`, `===`)
   - Check for hidden Unicode characters

### Medium Priority (P2)

6. Add language-aware scanning (detect non-English injection)
7. Add reversed text detection
8. Implement XML entity detection
9. Add RTL/LTR override detection
