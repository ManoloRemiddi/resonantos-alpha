Skills

Manage which skills each agent can use

What are skills? Skills are capability modules that give agents specific abilities, like GitHub operations, weather lookup, or video editing. Each skill has a `SKILL.md` instruction file.

How it works When a skill is enabled for an agent, OpenClaw includes that skill's description in the agent's prompt. The agent can then follow the skill's instructions when the task matches. When disabled, the agent doesn't know the skill exists.

How to use Click a toggle dot to enable or disable a skill for an agent. Purple dot means enabled, an empty circle means disabled. Changes save to `openclaw.json` immediately.

Default behavior If no skills are explicitly configured for an agent, it has access to all skills. Once you toggle any skill, that agent switches to an explicit allowlist with only the enabled skills.
