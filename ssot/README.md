# SSOT

Single Source of Truth documents and templates.

## Structure

| Directory | Purpose |
|-----------|---------|
| `L1/`, `L2/`, `L4/` | Actual SSOT documents (Cosmodestiny levels) |
| `templates/` | Templates for creating new SSOT documents |

## Templates

The `templates/` subdirectory contains:
- `L0/` — Foundation templates
- `L1/`, `L2/`, `L3/`, `L4/` — Level-specific templates
- `SSOT-STRUCTURE.md` — Document structure guide
